# E-commerce Customer & Sales Analytics

## Project Overview
This project analyzes customer behavior, sales trends, payment methods, and business insights using the Olist E-commerce Dataset.

The project was built using Python, SQL, Pandas, and Matplotlib to simulate real-world e-commerce analytics workflows.

---

## Technologies Used
- Python
- SQL
- Pandas
- NumPy
- Matplotlib
- SQLite
- Google Colab

---

## Key Analysis Performed
- Customer distribution analysis
- Revenue analysis
- Top product analysis
- Payment method analysis
- Monthly sales trend analysis
- SQL-based business queries

---

## Business Insights
- São Paulo had the highest customer concentration.
- Credit cards dominated payment transactions.
- Revenue contribution was concentrated among top-performing products.
- Monthly trends revealed fluctuations in customer purchasing behavior.
